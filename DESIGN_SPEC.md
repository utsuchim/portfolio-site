# ポートフォリオサイト設計書・仕様書

## 1. プロジェクト概要
個人のスキルや実績を紹介するためのポートフォリオサイト

### 目的
- 自己紹介とスキルセットの提示
- プロジェクト実績の展示
- 連絡先情報の提供

### 技術スタック
- **フロントエンド**: Next.js 14+ (App Router)
- **スタイリング**: Tailwind CSS
- **言語**: TypeScript
- **デプロイ**: Vercel

## 2. サイト構成

### 2.1 ページ構成
1. **トップページ (Home)**
   - ヒーローセクション
   - 自己紹介
   - スキル概要
   - プロジェクト一覧のプレビュー

2. **About（自己紹介）**
   - 詳細なプロフィール
   - 経歴・学歴
   - スキルセット詳細

3. **Projects（プロジェクト）**
   - プロジェクト一覧
   - 各プロジェクトの詳細ページ
   - 技術スタック、役割、成果物

4. **Contact（お問い合わせ）**
   - コンタクトフォーム
   - SNSリンク
   - メールアドレス

## 3. デザイン要件

### 3.1 デザインコンセプト
- モダンでクリーンなデザイン
- レスポンシブ対応（モバイルファースト）
- ダークモード対応
- アニメーション・トランジション効果

### 3.2 カラースキーム
- **プライマリカラー**: #3B82F6 (青)
- **セカンダリカラー**: #8B5CF6 (紫)
- **背景色（ライト）**: #FFFFFF
- **背景色（ダーク）**: #1F2937
- **テキスト（ライト）**: #1F2937
- **テキスト（ダーク）**: #F9FAFB

### 3.3 タイポグラフィ
- **メインフォント**: Inter
- **見出し**: 太字、大きめのサイズ
- **本文**: 通常、読みやすいサイズ

## 5. コンポーネント設計

### 5.1 共通コンポーネント
- **Header**: ナビゲーション、ロゴ、ダークモード切り替え
- **Footer**: コピーライト、SNSリンク
- **Button**: 汎用ボタンコンポーネント
- **Card**: プロジェクトカード、スキルカード
- **Section**: セクションラッパー

### 5.2 ページ固有コンポーネント
- **Hero**: トップページのヒーローセクション
- **SkillList**: スキル一覧表示
- **ProjectCard**: プロジェクトカード
- **ContactForm**: お問い合わせフォーム
